# AWS EC2 Web Server Project

## Project Overview

This project demonstrates how to deploy a static website on an Amazon EC2 instance using Apache HTTP Server.

---

## AWS Services Used

- Amazon EC2
- Security Groups
- Key Pair
- Amazon Linux 2023

---

## Technologies

- HTML
- Apache HTTP Server
- Linux
- SSH
- Git
- GitHub

---

## Project Steps

- Launch Amazon EC2 Instance
- Configure Security Groups
- Connect using SSH
- Install Apache HTTP Server
- Deploy HTML Website
- Access Website using Public IP
